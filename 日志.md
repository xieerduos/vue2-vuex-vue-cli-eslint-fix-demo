# 开发日志

## vue-cli 官网地址

https://cli.vuejs.org/zh/guide/creating-a-project.html#vue-create

## 执行下面命令

安装 @vue/cli

```
npm install -g @vue/cli
```

```
PS D:\Desktop\vue-demo> vue --version
@vue/cli 5.0.8
```

创建项目，项目名称是 hello-world

```
vue create hello-world

```

创建项目， . 是当前目录的意思

```
vue create .
```
